
# Repository layout

Top-level directories:

- `backend/`: FastAPI service, DB models, routers, scripts
- `frontend/`: React + Vite UI
- `helm/`: Helm chart assets
- `docs/`: (legacy) repository docs assets (screenshots, notes)

Documentation generated by MkDocs is located under `doc/` (this folder).

## Key files

- `docker-compose.yml`: local stack
- `ROADMAP.md`, `TODO.md`: project planning
- `ai_prompt.md`: development guidance / golden prompt
- `backend/app/scripts/probe_rgw_admin_contract.py`: non-mutating RGW contract probe (canonical `id`/`uid` checks)
