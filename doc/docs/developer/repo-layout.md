# Repository Layout

## Main directories

- `backend/`: FastAPI code, migrations, scripts, tests.
- `frontend/`: React UI, tests, build tooling.
- `doc/`: MkDocs documentation.
- `helm/`: Helm chart assets.
- `ops/`: operational helper scripts (cron jobs, etc.).

## Key files

- `docker-compose.yml`
- `docker-compose.build.yml`
- `doc/mkdocs.yml`
